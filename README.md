# ABC-Course-Project

### Постановка задачи

**Цель работы**: реализация и тестирование алгоритма решения СЛАУ на аппаратно-программной платформе CUDA.

**Объект исследования**: СЛАУ.

**Предмет исследования**: решения СЛАУ с помощью технологии CUDA.

**Задачи**:
Рассмотреть понятие СЛАУ и способы их решения. Проанализировать особенности решения систем линейных алгебраических уравнений на CUDA. Реализовать и протестировать алгоритмы решения СЛАУ на CUDA.


(Отчет в файле `Report.docx`)
